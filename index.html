Memory Functions Guide

MC

Memory Clear

Clears all stored values from memory register

MR

Memory Recall

Recalls the stored value from memory to display

M+

Memory Add

Adds current value to the memory register

M-

Memory Subtract

Subtracts current value from the memory register

MS

Memory Store

Stores current value in memory register

M

Memory View

Displays current memory value without recall

Scientific Functions Guide

Click the Scientific button to access advanced mathematical functions:

Trigonometric

  • sin, cos, tan - Basic trigonometric functions
  • sin⁻¹, cos⁻¹, tan⁻¹ - Inverse trigonometric functions
  • DEG/RAD/GRAD - Angle mode selection
  • hyp - Hypotenuse calculation

  • log - Base-10 logarithm
